# Rofi theme setup

Originally rofi theme was taken from [this repository](https://github.com/adi1090x/rofi). After some time while thinking about basic concept of my own rice (previously was on [swayfx](https://github.com/WillPower3309/swayfx)) changed theme according to simple black-which style which suits almost any wallpaper and theme.
